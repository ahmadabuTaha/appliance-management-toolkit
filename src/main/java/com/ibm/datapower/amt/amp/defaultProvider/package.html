A specific implementation of the AMP interface that defines a Java API for direct communication with DataPower devices. This implementation connects to the AMP endpoint of the XML Management Interface of the DataPower device.

The DataPower device implements an AMP server. Referring to the device implementation may resolve questions that this documentation does not make clear. This package implements AMP version 1.0 and uses the following schema and wsdl files from the DataPower device.

  • app-mgmt-protocol.xsd
  • app-mgmt-protocol.wsdl
